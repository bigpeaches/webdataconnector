Web Data Connector

The Web Data Connector is a developer extensibilitly platform that allows you, the developer, to bring the data you care about into Tableau.

This site is built for developers to get them up and running with the WDC API. The WDC SDK, including the simulator, developer samples, and documetion are all open source.
